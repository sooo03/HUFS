n개의 수가 주어졌을 때 이를 정확히 2개의 그룹 A, B로 나누어 A에 들어있는 수들의 합과 B에 들어있는 수들의 합이 일치하도록 하는 프로그램을 작성해보세요. 단, 주어진 n개의 수는 각각 A 또는 B 중 정확히 한 곳에만 속해야 하며, 각 그룹은 최소 1개 이상의 수를 포함하고 있어야 함에 유의합니다.

입력 형식

첫 번째 줄에는 n이 주어집니다.

두 번째 줄에는 n개의 수가 공백을 사이에 두고 주어집니다.

  • 2 ≤ n ≤ 100

  • 1 ≤ 주어지는 수 ≤ 1,000

출력 형식

그룹 A, B에 들어있는 수들간의 합을 동일하게 만드는 것이 가능하다면 "Yes", 불가능하다면 "No"를 출력합니다.

입출력 예제

예제1

입력:

4
2 2 1 5

출력:

Yes

예제2

입력:

5
4 7 8 9 5

출력:

No

예제 설명

첫 번째 예제에서 {2, 2, 1}, {5} 이렇게 2개의 그룹으로 나뉘면 그룹 내 합이 각각 5, 5가 되므로 두 그룹의 합을 같게 만들 수 있습니다.

두 번째 예제에서는 어떻게 나눠도 두 그룹의 합을 같게 만들 수 없습니다.

def can_balance_groups(nums):
    total_sum = sum(nums)
    if total_sum % 2 != 0:
        return "No"
    target_sum = total_sum // 2
    dp = [False] * (target_sum + 1)
    dp[0] = True
    for num in nums:
        for j in range(target_sum, num-1, -1):
            dp[j] |= dp[j - num]
    return "Yes" if dp[target_sum] else "No"
n = int(input())
nums = list(map(int, input().split()))
result = can_balance_groups(nums)
